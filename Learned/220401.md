### Shell Script

#### History
```
!! // 마지막 커맨드 실행
!n // n번째 커맨드
!-n // n번째 이전 커맨드
!$ // 마지막 커맨드의 마지막 파라미터
^old^new // 마지막 커맨드의 old 파라미터를 new로 교체하여 실행

```
#### 반복문
```
touch {0..10}.txt 
// create 0.txt, 1.txt, 2.txt .... 10.txt 

touch {0..10..2}.txt 
//create 0.txt, 2.txt, 4.txt .... 10.txt
```
